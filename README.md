##  Getting Started

```
# 1. Clone the repo
git clone https://github.com/SahilCharan/Link-Saver-app.git
cd Link-Saver-app

# 2. Install dependencies
npm install

# 3. Setup environment variables
cp .env.example .env
# Fill in your Supabase URL & Anon Key in `.env`

# 4. Start the dev server
npm run dev
```
